# 프로젝트 소개글

**CineMatch**는 MovieLens 1M 데이터를 활용해 만든 다단계 개인화 추천 시스템입니다.  
단순 인기순 추천이 아니라 **협업 필터링 기반 후보 생성 → 사용자/아이템 메타데이터 기반 재랭킹 → 다양성 리랭킹(MMR)** 구조를 적용해 추천 정확도와 추천 목록의 다양성을 함께 개선하도록 설계했습니다.

프로젝트에서는 데이터 다운로드와 전처리, 학습 파이프라인, 오프라인 평가 리포트, FastAPI 기반 서빙, HTML 대시보드, 피드백 로그 수집까지 한 번에 구현했습니다.  
추천 시스템 포트폴리오에서 자주 빠지는 **콜드스타트 처리**, **Catalog Coverage / Diversity / Novelty / Personalization** 같은 운영형 지표까지 포함해 “모델 한 개 학습”이 아니라 **실무형 추천 시스템** 관점으로 정리한 것이 강점입니다.
