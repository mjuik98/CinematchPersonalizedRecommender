# 이력서 bullet 예시

- MovieLens 1M 기반 다단계 개인화 추천 시스템 설계 및 구현
- 협업 필터링 후보 생성(TruncatedSVD)과 메타데이터 기반 랭커(Logistic Regression)를 결합한 multi-stage recommendation pipeline 구축
- HitRate@K, NDCG@K, MRR 외 Coverage, Novelty, Diversity, Personalization 지표를 포함한 오프라인 평가 자동화
- FastAPI 기반 추천/콜드스타트/유사 아이템/피드백 API와 HTML 대시보드 구현
- SQLite 로그 적재 및 analytics summary API를 통해 추천 요청/피드백 추적 기능 구현
- Docker, unittest, GitHub Actions를 포함한 재현 가능한 제출형 ML 서비스 패키지 구성
